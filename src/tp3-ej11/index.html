Ejercicio 11

Demostrar que dados \( \color{red}{a} \) y \( \color{red}{b} \) en \( \color{red}\mathbb{Q} \) tales que \( \color{red}{a \lt b} \), existe \( \color{red}{x \in \mathbb{Q}} \) tal que \( \color{red}{a \lt x \lt b} \)

Gracias totales

Vamos de verdad...

  • Primero: tenemos que ver que existe este número racional
  • Segundo: tenemos que verificar que efectivamente está entre a y b

Sean \( a, b \in \mathbb{Q} \), tales que existen \( p, q, n, m \in \mathbb{Z} \) con \( q \ne 0 \) y \( m \ne 0 \)

$$a = \frac{p}{q}, \quad b = \frac{n}{m}$$

$$a + b = \frac{p}{q} + \frac{n}{m}$$

$$= \frac{p \cdot m}{q \cdot m} + \frac{n \cdot q}{m \cdot q}$$

$$= \frac{pm}{qm} + \frac{nq}{qm}$$

$$= \frac{pm + nq}{qm}$$

$$\in \mathbb{Q}, \quad \text{ya que } pm + nq \in \mathbb{Z},\ qm \in \mathbb{Z} \setminus \{0\}$$

$$\text{Como la suma y el producto son operaciones cerradas en } \mathbb{Z},$$

$$\text{entonces el resultado pertenece a } \mathbb{Q}$$

Entonces

$$a + b = \frac{pm + nq}{qm}$$

$$\frac{a + b}{2} = \frac{1}{2} \cdot \frac{pm + nq}{qm}$$

$$= \frac{pm + nq}{2qm}$$

$$\text{Numerador: } pm + nq \in \mathbb{Z}$$

$$\text{Denominador: } 2qm \in \mathbb{Z} \setminus \{0\}$$

$$\Rightarrow \frac{pm + nq}{2qm} \in \mathbb{Q}$$

$$\text{La división de un número racional entre un entero distinto de cero sigue siendo un número racional.}$$

Resta ver que x está entre a y b

$$a = \frac{a}{1}$$

$$= \frac{2a}{2 \cdot 1}$$

$$= \frac{a + a}{2}$$

$$= \frac{a}{2} + \frac{a}{2}$$

$$\lt \frac{a}{2} + \frac{b}{2} \quad \text{(ya que } a \lt b \text{)}$$

$$= \frac{a + b}{2}$$

$$\therefore \quad a \lt \frac{a + b}{2}$$

$$b = \frac{b}{1}$$

$$= \frac{2b}{2 \cdot 1}$$

$$= \frac{b + b}{2}$$

$$= \frac{b}{2} + \frac{b}{2}$$

$$> \frac{a}{2} + \frac{b}{2} \quad \text{(ya que } a \lt b \text{)}$$

$$= \frac{a + b}{2}$$

$$\therefore \quad b > \frac{a + b}{2}$$

$$\therefore$$

$$a \lt \frac{a + b}{2} \lt b$$

$$\Rightarrow \exists \, x = \frac{a + b}{2} \in \mathbb{Q} \quad \text{tal que} \quad a \lt x \lt b$$
